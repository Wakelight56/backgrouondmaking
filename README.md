# 表情符号背景生成器

一个类似iOS的DIY表情符号背景生成器网站，允许用户上传图片并自定义背景颜色。

## 功能特性

- 📁 支持上传多张图片（点击或拖放）
- 🎨 自定义背景颜色
- 🔄 一键应用颜色到所有图片
- 👀 实时预览效果
- 💾 单独下载或批量下载图片
- 📱 响应式设计，适配不同屏幕尺寸
- 🎯 现代化iOS风格界面

## 如何使用

### 本地使用
1. 克隆仓库：`git clone https://github.com/Wakelight56/backgrouondmaking.git`
2. 进入目录：`cd backgrouondmaking`
3. 启动本地服务器：`python -m http.server 8000`
4. 在浏览器中访问：`http://localhost:8000`

### GitHub Pages 部署

1. 打开GitHub仓库：https://github.com/Wakelight56/backgrouondmaking
2. 点击 "Settings" 选项卡
3. 在左侧菜单中选择 "Pages"
4. 在 "Build and deployment" 部分，将 "Source" 设置为 "Deploy from a branch"
5. 选择 "main" 分支（或 "master" 分支），然后点击 "Save"
6. 等待几分钟，GitHub Pages 将会部署网站
7. 部署完成后，你将看到访问链接，格式为：`https://wakelight56.github.io/backgrouondmaking`
